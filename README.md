# TicTacToe
A Tic Tac Toe game written in C

To execute this first clone the repository

```git clone https://github.com/slayeh17/TicTacToe```

Make sure you have ```GCC``` installed.

Now to run the code

```
cd TicTacToe
gcc TicTacToe.c
./a //For Windows
./a.out //For Linux
```
